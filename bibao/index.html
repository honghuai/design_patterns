闭包的概念

(1)定义了一个wrap函数
(2)在A中定义了showNum函数
(3)在wrap中返回showNum（在wrap中返回showNum的引用）
(4)执行wrap(),把wrap的返回结果赋值给变量 c
(5)执行 c()
闭包的定义：
当一个内部函数被其外部函数之外的变量引用时，就形成了一个闭包。

例子：

    function wrap() {
        var count = 0;
        function showNum() {
            count ++;
            console.log("num = " + count);
        }
        return showNum;
    }
/*wrap中的count一直保存在内存中*/
    wrap()();//1
    var c = wrap();
    c();//1
    c();//2
    c();//3
    var d = wrap();
    d();//1
